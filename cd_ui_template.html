Quick Reference
Running an extraction
Request → select file → SME validates → extract → store. Each step is logged automatically. Flagged SME decisions are recorded but do not proceed to extraction — the SME must re-select a file.
SME audit trail
Every decision (approve, flag, approve with notes) is timestamped and written to the SME_Audit_Log sheet. Flags are preserved even when a file is later re-selected and approved — giving a complete change history per project.
Where data is stored
Three sheets in Client_ROI_Tracker.xlsx: All_ROI_Data (extracted values), SME_Audit_Log (validation checkpoints), Source_File_Log (file references with version, path, and modification date).
Building dashboards
Use the Dashboards view to build cross-publisher or cross-year comparisons. Choose a preset template or use custom filters to select any combination of clients, publishers, and years.
